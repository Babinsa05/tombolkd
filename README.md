# Udayana Portal PWA

Progressive Web App untuk akses cepat portal Udayana dengan dukungan APK.

## Fitur

✅ **PWA Full Support** - Install sebagai aplikasi native  
✅ **Service Worker** - Bekerja offline  
✅ **APK Download** - Versi Android native  
✅ **GitHub Pages** - Hosting gratis  
✅ **Responsive Design** - Mobile & Desktop  
✅ **Fast Loading** - Optimized performance  

## Instalasi

### 1. Sebagai PWA (Recommended)
1. Buka https://babinsa05.github.io/tombol2/
2. Klik tombol "Install PWA" di kanan bawah
3. Konfirmasi instalasi
4. Aplikasi akan muncul di home screen

### 2. Sebagai APK (Android)
1. Klik tombol "Download APK"
2. Izinkan instalasi dari sumber tidak dikenal
3. Buka file APK yang diunduh
4. Instal aplikasi
